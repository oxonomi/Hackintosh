# Hackintosh
OpenCore:	0.9.1
macOS: 	Ventura 13.3.1
Windows:	10
Linux:		Debian

## Hardware	
- Mobo:		Gigabtye Z490m Gaming X
- CPU:		Intel i5-10600k
- GPU:		Sapphire RX 5600 XT
- RAM:		Corsair Vengeance 2x16GB 3200
- Wifi + BT:	Fenvi T919
- Storage:	Crucial P1 1TB (macOS), Crucial MX500 1TB (Windows), Crucial MX500 1TB (Linux)
